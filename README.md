# Mikroupravljači — ATmega328P

Materijali za programe cjeloživotnog učenja iz područja **Mikroupravljači**, uključujući skeletni kod vježbi, potpuna rješenja i udžbenik.

## Sadržaj

| Modul | Vježbe |
|-------|--------|
| Digitalni izlazi | vjezba41 – vjezba47 |
| Digitalni ulazi | vjezba51 – vjezba56 |
| LCD displej | vjezba61 – vjezba67 |
| ADC | vjezba71 – vjezba73 |
| Tajmeri | vjezba91 – vjezba96 |
| PWM | vjezba101 – vjezba104 |
| UART | vjezba111 – vjezba116 |
| EXTINT | vjezba121 – vjezba122 |
| Moduli (integrirane vježbe) | vjezba131 – vjezba136c |
| Ostalo | vjezba141 – vjezba145 |

Svaki modul sadrži dvije mape:
- **`vjezbe/`** — skeletni kod za izradu vježbi
- **`rjesenja/`** — potpuna rješenja

## Hardver

- Mikroupravljač: **ATmega328P** (Arduino Uno kompatibilan)
- IDE: **Atmel Studio**
- Jezik: **C++** (AVR dialekt)
